# `JSON`

By NriotHrreion

> A wheel of json compiler

## Usage

```js
JSON.parse(`
{
    "abc": 123,
    "test": "Hello",
    "ok": true,
    "props": null
}
`); // { abc: 123, test: 'Hello', ok: true, props: null }
```

## Development

```bash
# Clone repo
git clone https://github.com/NriotHrreion/json
# Install dependencies
npm i
# Run dev environment
npm run dev
# Run tests
npm run test
```
